近代物理实验

发表于 2025-03-17 更新于 2025-11-08 分类于 学习&科研 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 5 分钟

\(\qquad\!\!\)考虑到保研要求的前六个学期修完的专业基础课和专业核心课的毕业最低学分要求中将“近代物理实验 Ⅱ”课程除外，我便将“近代物理实验Ⅰ”课程放在了2024-2025学年春季学期选修。本学期共需要进行七次实验，除第一周选择实验与五一假期外，其余十四周每两周选做了一次实验。除了第一次实验——塞曼效应 是因为与选课班级的老师绑定几乎强制选择的以外，其他的六个实验基本上都是在收集了各方同学的建议后精挑细选出来的相对正常一些的实验，在这里也可供大家参考。（还是忍不住要吐槽一下北京大学基础物理实验教学中心的选修实验系统，实在是太垃圾了到处都是 bug，第一次抢实验真的容易被干懵然后与最理想的实验序列失之交臂。）

\(\qquad\!\!\)近代物理实验的实验报告要求较为严格，要求写作为研究论文型实验报告，需按照一定的格式要求组织内容，尤其强调进行分段叙述的重要性，在一定程度上对于“未来”的科研论文写作打下相应的基础。不知道为什么本学期的初始文件中并没有包括 \(\mathrm{\LaTeX}\) 模版，我在 GitHub 上搜索找到了 PKUMpLtX 项目——A LaTeX document class for 'Modern Physics Laboratory' in PKU based on revtex4-2 并使用该文档类最新版本 2.1.6 进行了实验报告的写作。我将老师上课讲解和发邮件提醒的实验报告写作要求整理到了 \(\mathrm{\LaTeX}\) 实验报告模版上，作为自己撰写实验报告的基本依照，因为本人对于格式的一些强迫的完美主义倾向，在具体的实验报告中显得可能就会有些八股，在这里与具体的实验报告一同给出均仅供大家参考：

  • 近代物理实验报告模版

  • 塞曼效应

    老师面批后给出的建议：

    • 可以再画一张能级分裂图。
    • 建议给出具体的利用压强计算波数差的理论公式。
    • 前面的理论部分给出后面会涉及到的物理量的定义即可，无需这么详细地把书都抄上来。
    • 不要直接把参考文献的引用引用到某节的标题上。

  • He-Ne 气体激光器放电条件的研究

    • 摘要中不要涉及到缩写。
    • 摘要中的重要结论要具体说出来。
    • 实验装置只用一张图就可以，有从属关系的建议合并到一张图上。
    • 实验过程的详细记录无需落到实验报告上。
    • 图 4 作为最重要的结论，可以有如下改进：
      • 不仅不同的数据用不同的颜色，散点图中的散点也可以用不同的符号。
      • 表示最大值的特征点可以特别标注出来。
      • 特征点相关物理量的变化趋势可以在图中（用箭头等）标注出来。
    • 尽量分段叙述优于说明书式以编号逐条叙述。

  • 扫描电子显微镜

    • 图片的位置不用拘泥于第一次引用的段落之后，如果因为浮动体浮动到不太恰当的位置，也可以置于段前。
    • 实验装置只用一张图就可以，比较并列或逻辑上的关系可以用子图合并到一张图上。

  • 非线性对流斑图

  • 铁磁共振

    • 摘要里最好给出典型实验数据结果。
    • 图 1 的 caption 应更改为“观测铁磁共振的实验装置图“。
    • 图 2 中基线不平，小磁场数据有点少。
    • 结论需要添加一些实验的结果与讨论， 比摘要更详细。
    • 思考题一结论有误，应该为“样品可以放在腔内微波磁场最大、微波电场为零的位置，即 \(x=a/2\)，\(y\) 任意，\(z=kl/8,k=1,2,\cdots,7\) 的位置上．”。

  • 磁光克尔效应

  • 热噪声与散粒噪声实验

\(\qquad\!\!\)以上实验报告均使用 MpLtX 文档类，由 LaTeX 排版，使用 TeX Live 2025 发行版，XeLaTeX 编译器（格式+引擎）于 PKUTEX 平台生成。仅供参考，如因此产生的任何问题，本人概不负责😈。
