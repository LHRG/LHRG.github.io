题目链接：单身排查

关于题目：作为 T1，一道很水很水的有关位运算的题目。

应用算法：无。

\(\ \ \ \ \ \ \)一句话题意：一共 \(n\) 个数，其中的存在某一个数字出现了奇数次，其它的数字均出现偶数次，请你求出这个数字。

\(\ \ \ \ \ \ \)由于太水了，我就直接说了：C++ 的位运算中有一个东西叫异或（^），相同的偶数个数字的异或和为 \(0\)，相同的奇数个数字的异或和为该数字。

\(\ \ \ \ \ \ \)所以这道题就相当于求出所有数字的异或和。
