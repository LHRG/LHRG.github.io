    非常初步的立直的读牌

    发表于 2025-09-07 分类于 日本麻将 阅读次数： 本文字数： 1.1k 阅读时长 ≈ 4 分钟

    \(\qquad\!\!\)读《统计学麻雀战术》又读到和一些基本的立直的读牌相关的部分，想到在《79博客》中有两小节相关的内容似乎可信度不是很高，于是想整体搜集一遍相关的最基本的资料整理出来，算是对于立直的读牌非常初步的学习，最后总结为《统计学麻雀战术》中的两节+几条小结论。

    来自《科学麻将》

    • “因为是立直宣言牌周边的牌所以危险”这句话未必是对的，立直宣言牌的旁边与普通的危险牌危险度一样。
    • 里筋并不危险，里筋与普通的危险牌的危险度一样。
    • 间四间的危险度只会提高那么一点点。

    来自《用数据读牌》

    • “仅凭是里筋这一条理由就要作出必要程度以上的警戒”这个结论是错误的。
    • 很早切出5时对应作为里筋的14和69需要警戒。
    • 间四间要比一般的无筋牌在危险度上要略微上升的考量。

    来自《79博客》

    • 面对早巡切出了数牌5的立直，就应该警戒数牌5对应花色的14和69。
    • 间四间也比较危险。
    • 被拆掉的两面搭子的里筋：拆掉两面搭子的立直，要小心其二度进张。
    • 拆掉嵌张搭子的立直的跨筋是危险的。
    • 切安全牌立直，宣言牌的前一张牌可以看做和铳牌没有关联，切掉安全牌的立直，是由两个固定的两面搭子来的，几乎不会有双碰进张。
    • 逆切的跨筋：拆掉嵌张搭子的立直，逆切的跨筋安全度较高。

    来自《统计学麻雀战术》

    专题二十四 注意切【1饼】听靠张【2 饼】的情况

    1. 对于无筋3-5，并没有因为是立直宣言牌周边的牌所以危险，反而会更加安全一些。
    2. 立直宣言牌周边牌危险度会上升的情况：
      1. 切19立直的28：
        • 无筋28铳率10.5%超过无筋37
        • 筋28铳率7.2%超过筋37～无筋19
      2. 切28立直的19：
        • 无筋19铳率9.6%～无筋37
        • 筋19铳率5.5%～筋37
      3. 切37立直的19：
        • 无筋19铳率8.4%～无筋28
        • 筋19铳率3.3%～筋28
      4. 类间四间：切过37切82立直的64：
        • 铳率9.4%～无筋37
    3. 切字牌立直时筋牌会变安全。

    专题二十五 对于立直家切掉的【5 饼】有所反应

    1. 筋28的危险度切过0相对于切过5要稍微低一些。
    2. 切5立直的筋28引挂的铳率还是比筋28高～筋37。
    3. 切过0筋37和半筋A46的铳率明显降低，比筋28还要安全。
    4. 无筋19和半筋B46要比正常情况稍微危险一点。
    5. 立直前同时切过5/0和19时，筋2837的铳率1.5～筋19。

    总结一下：

    来自《统计学麻雀战术》的结论+如下：

    1. 立直宣言牌的旁边与普通的危险牌危险度基本相同甚至更加安全，在找安牌时更应该根据牌河的其他信息排除其危险。
    2. 里筋并不危险，里筋与普通的危险牌的危险度一样。
    3. 间四间的危险度确实会提高一点点，大约1.2倍左右。
    4. 5&0的里筋49确实是大家都会提到需要注意的危险牌，要比正常情况稍微危险一点但是也没有到上升一个等级的情况。
    5. 被拆掉的两面搭子的里筋：拆掉两面搭子的立直，要小心其二度进张。
    6. 顺切的跨筋：拆掉嵌张搭子的立直，顺切的跨筋危险度较高。
    7. 逆切的跨筋：拆掉嵌张搭子的立直，逆切的跨筋安全度较高。
    8. 切安全牌立直，宣言牌的前一张牌可以看做和铳牌没有关联，切掉安全牌的立直，是由两个固定的两面搭子来的，几乎不会有双碰进张。
